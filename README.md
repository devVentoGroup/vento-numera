# NUMERA

NUMERA es la app Vento OS para inteligencia economica operativa.

Responsabilidad inicial:

- Centros de costo.
- Gastos por sede y satelite.
- Punto de equilibrio.
- Margen y rentabilidad.
- Presupuesto vs real.
- Costo teorico vs costo real.

Regla de arquitectura: Numera consolida y analiza; no reemplaza los flujos operativos de Origo, Nexo, Fogo, Pulso, Pass o Anima. Las migraciones de base de datos viven en `vento-shell`.
